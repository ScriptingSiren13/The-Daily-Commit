# Day 37 — Python Revision: Strings

Today I revised **Strings in Python**, focusing on how strings work internally and the most common operations used in real coding.

---

##  Topics Revised

### **1) What Strings Are**
- Strings are sequences of characters
- Strings are **immutable** (cannot be changed in-place)

---

### **2) Creating Strings**
- Single quotes: `'hello'`
- Double quotes: `"hello"`
- Triple quotes: `"""multi-line"""`

---

### **3) Indexing & Slicing**
- Accessing characters using indexes
- Slicing substrings using ranges
- Using negative indexing

---

### **4) Common String Methods**
Revised frequently used methods like:

- `.lower()`, `.upper()`, `.title()`
- `.strip()`, `.lstrip()`, `.rstrip()`
- `.replace()`
- `.split()`, `" ".join()`
- `.find()`, `.count()`
- `.startswith()`, `.endswith()`

---

### **5) String Formatting**
Revised the main formatting styles:

- f-strings
- `.format()`

---

### **6) Useful Concepts**
- Checking length using `len()`
- Membership: `'a' in text`
- Iterating through strings
- Comparing strings
